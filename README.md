# Stanfliet OTA Energy System

A full-stack energy management system built with React, Node.js, and Vercel.

## Features
- Real-time energy monitoring
- MQTT integration
- Interactive maps with Leaflet
- PDF report generation

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
